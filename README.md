# Exploratory Data Analysis (EDA) - Chocolate Sales

**Цель проекта:** Ответить на 3-5 простых бизнес-вопросов к данным с помощью SQL и визуализации.

**Источник данных:** [Kaggle - [ссылка на датасет](https://www.kaggle.com/datasets/saidaminsaidaxmadov/chocolate-sales)]

**Используемый стек:**
- SQL (для запросов)
- Excel / Google Sheets (для первичного анализа)

**Вопросы для исследования:**

1.  **Динамика продаж**  
    Как менялась выручка по месяцам? Где пики, где провалы?
2.  **Топ продуктов и стран**  
    Какие 5 продуктов и 3 страны приносят больше всех денег? Выведи рейтинг.
3.  **Эффективность продавцов**  
    Кто топ-5 продавцов по выручке? Какую долю рынка они отжимают?
4.  **Анализ среднего чека**  
    С какой коробки шоколада мы больше всего зарабатываем? Где самая высокая цена за штуку?
5.  **Сезонность по продуктам**  
    Для трёх самых продаваемых продуктов: в какие месяцы продажи были выше среднего?

**Структура репозитория:**
- `/data` - исходный файл данных (.csv)
- `/sql` - SQL-запросы
- `/results` - выводы, скриншоты
